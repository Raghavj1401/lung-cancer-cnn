In this project, i have built a model that can detect lung cancer using CT scans. 
CNN is used for this project.
CT scan images are classified into four categories:

- adenocarcinoma
- large_cell
- normal
- squamous

I have used:
tensorflow
python
numpy
matplotlib
streamlit

Workflow:
Dataset - CNN Training - Model - Streamlit App

The trained model (lung_cancer_cnn.h5) is not included in this repository because it exceeds GitHub's limit.

Place the model file here before running the app:

models/lung_cancer_cnn.h5
